# Dataset Information

## Dataset Overview

This project uses multiple datasets related to users, restaurants, menus, food items, and customer orders to analyze sales performance and customer behavior on the Zomato platform.

## Files Included

- food.xlsx
- menu.xlsx
- orders.xlsx
- orders_Type.xlsx
- restaurant.xlsx
- users.xlsx

## Analysis Areas

- Sales Analysis
- Customer Analysis
- User Retention
- City Performance
- Restaurant Performance
- Food Category Analysis

## Purpose

The datasets were used to build an interactive Power BI dashboard that provides insights into sales trends, customer engagement, and operational performance.
