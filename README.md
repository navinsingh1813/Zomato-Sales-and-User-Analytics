# 🍽️ Zomato Sales & User Analytics

![Dashboard Main](https://github.com/navinsingh1813/Zomato-Sales-and-User-Analytics/blob/main/Images/Main.png)

## Project Overview

Zomato Sales & User Analytics is a Business Intelligence project developed using Power BI to analyze sales performance, customer behavior, user retention, restaurant operations, and city-level business trends.

The project transforms raw operational data into interactive dashboards that support data-driven decision-making.

---

## Business Objective

The objective of this project is to understand customer behavior, monitor business performance, and identify growth opportunities through sales and user analytics.

Key areas analyzed include:

- Sales Performance
- Customer Acquisition
- User Retention
- City-wise Performance
- Restaurant Operations
- Ratings Analysis
