# Business Problem

Zomato operates across thousands of restaurants and cities, generating large volumes of customer, order, menu, and restaurant data.

The objective of this project is to analyze sales performance, customer behavior, user engagement, and city-level business trends to identify growth opportunities and support data-driven decision-making.

Key focus areas include:

* Sales Performance Analysis
* Customer Acquisition & Retention
* User Behavior Analysis
* City-wise Performance
* Restaurant Network Analysis
* Food Category Performance
* Order Trends & Growth
* Customer Ratings Analysis

The insights generated from this analysis can help stakeholders improve customer engagement, optimize operations, and identify high-performing markets.
