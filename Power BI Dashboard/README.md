# Power BI Dashboard

## Dashboard Overview

This dashboard analyzes Zomato's business performance across sales, customers, users, restaurants, and cities.

Dashboard Sections:

### Dashboard Overview
- Sales KPIs
- Order Metrics
- Top Cities
- Yearly Sales Trend

### User Performance
- Active Users
- Customer Acquisition
- Customer Retention
- User Demographics

### City Analysis
- City-wise Sales
- Ratings Analysis
- Active User Analysis
- Customer Growth

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

## Outcome

The dashboard helps identify customer trends, operational performance, and business growth opportunities.
